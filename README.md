# 🍽️ NearEat  
**“지금 이 근처 뭐 먹지?” — 위치 기반 즉시 음식점 추천 앱**

SwiftUI 기반으로 제작된 NearEat은  
현재 위치를 기반으로 음식점을 검색하고,  
카테고리 / 키워드 / 랜덤 추천 기능을 통해  
사용자가 빠르게 식당을 선택할 수 있도록 돕는 앱입니다.

---

## 🚀 주요 기능 (Features)

### 🔍 1. 키워드 기반 음식점 검색
- 사용자가 입력한 검색어(query)를 기반으로 주변 음식점을 탐색
- 예시: `"치킨"`, `"라멘"`, `"카페"`

### 📍 2. 현재 위치 기반 추천
- CoreLocation을 통해 사용자 GPS 정보를 받아오고  
  Kakao Local API로 주변 음식점 목록을 가져옴

### 🍱 3. 7개 카테고리 선택 (3줄 Grid UI)
- 한식, 양식, 일식, 중식, 카페, 분식, 전체  
- 선택 즉시 자동 검색 실행  
- 선택된 카테고리 하이라이트 표시

### 🎲 4. 랜덤 추천 기능
- 검색된 음식점 목록에서 1개를 랜덤으로 선택해 카드로 보여줌

### 🔗 5. 카카오 상세 페이지 연결
- 음식점 클릭 시 `place_url`을 Safari로 오픈

## 🧠 기술 스택 (Tech Stack)

| 기술 | 설명 |
|------|------|
| **SwiftUI** | 전체 UI 개발 |
| **MVVM 구조** | View / ViewModel / Model 분리 |
| **CoreLocation** | GPS 기반 위경도 수집 |
| **URLSession** | Kakao API 네트워크 통신 |
| **Kakao Local API** | 음식점 정보 조회 |
| **GitHub** | 형상 관리 |
